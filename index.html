Política de Privacidad

Esta política de privacidad aplica a la aplicación Frase Diaria, desarrollada por Studio Santidark.

Recopilación de datos

La aplicación no recopila información personal identificable del usuario. Únicamente se genera y utiliza un token anónimo proporcionado por Firebase Cloud Messaging (FCM) para el envío de notificaciones push.

Publicidad

La app muestra anuncios mediante Google AdMob. Este servicio puede utilizar identificadores del dispositivo (como ID de publicidad) para mostrar anuncios personalizados o no personalizados. Consulta la política de privacidad de Google para más información:

  • https://policies.google.com/privacy

Conexión a Internet

La aplicación requiere conexión a Internet para cargar contenido y mostrar anuncios. El uso de datos móviles o Wi-Fi queda bajo responsabilidad del usuario.

Seguridad

La app no almacena ni transmite datos personales. Se toman medidas para asegurar el uso responsable de los servicios de terceros.

Enlaces a terceros

La app puede contener enlaces o mostrar contenido de servicios externos, como anuncios de Google. No tenemos control sobre sus políticas o contenidos.

Cambios a esta política

Podemos actualizar esta política en cualquier momento. Se recomienda revisar esta página periódicamente.

Contacto

Si tienes dudas o sugerencias, puedes contactarnos en studiosantidark@gmail.com.
